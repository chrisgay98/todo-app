# To-Do List App (Python)

## Overview
A simple command-line To-Do List application built with Python.  
This project demonstrates lists, loops, functions, input handling, and error handling.

## Features
- Add tasks
- View tasks
- Delete tasks
- Quit the app
- Input validation and error handling
